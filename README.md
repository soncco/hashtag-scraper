# Hashtags scraper
Get hashtags for trendogate.com

## Requirements.

* Last version of node.js.
* A Google Firebase account.

## Installation.

```
$ npm i
```

## Use

### Scraping

```
$ node scraping.js
```

### Ocurrences

```
$ node ocurrences.js
```

### First 100 hashtags ordered by ocurrences.

```
$ node sort.js
```
